# caozha-whois（PHP）

caozha-whois，是一个采用原生PHP写的域名Whois查询模块，支持全球大部分域名的whois查询，支持中文域名在内的多种域名后缀，包括：.com，.net，.cn，.com.cn，.org，.biz，.cc，.mobi，.中国，.公司等等几百种域名后缀。
